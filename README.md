# GTeams

Sistema de Gestão de Equipes, Metas e Colaboradores

---

## Descrição

O GTeams é uma solução de API completa para gestão de equipes e colaboradores. Desenvolvido em GO, o sistema possibilita o acompanhamento e gerenciamento de times, metas mensais, colaboradores e medições de desempenho de forma centralizada e eficiente.
